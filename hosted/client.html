Pokémon Database

Retrieves a list of the names of each Pokemon based on the three possible parameters provided:

  • Name. Ex. "Gengar"
  • Type (comma separated). Ex. "ghost, poison"
  • National Pokédex Number. Ex. "94"

Note that the type search is inclusive rather than exclusive. If multiple types are entered without an accompanying name or Pokédex number, any Pokémon that contain a type satisfying the requirements will be added to the search results.

Also keep in mind this will only be returning the name of each Pokémon, rather than all of their accompanying data.

Note that the type search is inclusive rather than exclusive. If multiple types are entered without an accompanying name or Pokédex number, any Pokémon that contain a type satisfying the requirements will be added to the search results.

Retrieves a list of Pokemon based on the national Pokédex number provided. For example, a number of 94 would return Gengar as a search result. All information about the Pokémon will be returned by this request.

Retrieves a list of all Pokémon available in the database. Much like with the getPokémon request, this will include all available information about said Pokémon.

Adds another Pokémon to the current database. Since the data is stored locally on the server, it will disappear anytime the server restarts. On Heroku, this is usually every 30 minutes or so.

Adding another Pokémon requires all of the following parameters:

  • National Pokédex Number (ie. 94)
  • Name (ie. Gengar)
  • Image of the Pokémon (ie. http://www.serebii.net/pokemongo/pokemon/094.png)
  • Pokémon type(s), comma separated if it has multiple (ie. Ghost, Poison)
  • Height (ie. 1.50 m)
  • Weight (ie. 40.5 kg)
  • Weakness/Weaknesses, comma separated if it has multiple (ie. Ground, Psychic, Ghost, Dark)
  • Next evolution, if applicable (ignore if none)

Note that only the national Pokédex number, name, and Pokémon types are required parameters and will return an error if not filled out. All other parameters are optional.

Allows user to assign a tier to any Pokémon in the database, which will then be added to the displayed search results. The required parameters will be as follows:

  • National Pokédex number of Pokémon (ie. 94 for Gengar)
  • Pokémon tier placement (ie. RU, or Rarely Used, in Pokémon Scarlet and Pokémon Violet)

The tiers provided are Smogon's, the competitive standard for single battles, and based off the most recent Pokémon games as of October 2024, being Pokémon Scarlet and Pokémon Violet. You can search Smogon's database for tier placementshere.
